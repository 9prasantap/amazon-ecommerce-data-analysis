# Amazon E-Commerce Data Analysis

This project focuses on analyzing an Amazon-style e-commerce dataset using SQL, Python, and Power BI.

The objective is to explore sales performance, customer purchasing behavior, product trends, seller performance, inventory management, and operational efficiency through data-driven analysis.

The project is being built as an end-to-end analytics workflow, starting with SQL-based business analysis and gradually expanding into Python-based analysis and Power BI dashboarding.

### Customer Segmentation

* Customer Spending Segments

### Advanced Business Analysis

* Product Profit Margin Analysis
* Most Returned Products

### Pareto Analysis

* 80/20 Revenue Contribution Analysis

* Monthly Sales Trends
